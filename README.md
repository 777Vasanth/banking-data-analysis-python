Banking Data Analysis using Python

## Project Description

This project performs end-to-end data analysis on a simulated banking dataset using Python. It focuses on extracting meaningful business insights from customer, account, transaction, loan, investment, and credit score data.

The analysis answers 50+ real-world business questions such as:

* How many accounts does each customer hold?
* Which customers have the highest balances?
* What is the distribution of credit scores?
* Which customers have high loans but low credit scores?
* Investment behavior across different cities
* Transaction patterns and customer activity

The project uses **Pandas** for data manipulation and **Matplotlib** for visualization.



## Objective

To simulate real-world banking analytics and demonstrate data analysis skills including:

* Data cleaning and transformation
* Aggregation and grouping
* Data merging across multiple datasets
* Business insight generation
* Data visualization



##Key Insights

* Identified high-value customers based on balance and transactions
* Detected customers with financial risk (high loan + low credit score)
* Analyzed city-wise income and investment patterns
* Compared credit vs debit transaction trends
* Found customers with multiple account types and investment behaviors



## Tools & Technologies
